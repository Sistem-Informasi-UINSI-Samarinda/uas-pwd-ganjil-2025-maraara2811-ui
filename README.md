# uas-pwd-ganjil-2025-maraara2811-ui

## Deskripsi
Proyek ini merupakan tugas UAS Pemrograman Web Dasar (PWD) Semester Ganjil 2025.
Aplikasi dibuat menggunakan PHP Native, HTML, CSS, dan MySQL (XAMPP).

## Fitur
- Halaman utama
- Autentikasi (login & register)
- Manajemen data
- Upload file

## Struktur Folder
- assets/
- auth/
- config/
- includes/
- pages/
- uploads/
- index.php

## Cara Menjalankan
1. Jalankan XAMPP
2. Aktifkan Apache & MySQL
3. Letakkan folder proyek di `htdocs`
